### Stack Tecnologico Real

- Backend: Python 3.11+, http.server (stdlib), sem framework web
- Frontend: HTML + CSS + Vanilla JS (sem React/Vue/Angular no dashboard)
- Autenticacao: JWT (PyJWT, HS256, 24h expiry) + API Keys (SHA-256)
- Senhas: bcrypt (custo 12) com fallback SHA-256
- Banco: JSON file-based (~/.hermes/shared_data/) com fallback opcional para Neon PostgreSQL
- Pagamentos: Stripe (checkout, subscriptions, webhooks, portal) - opcional
- Pipeline: OpenStreetMap Overpass API para extracao de leads
- Propostas: Claude AI via CLI "hermes" OU template hardcoded no generate_proposals_direct.py
- Automacao: n8n (workflows nao versionados/exportados no repo)
- Deploy: Render, Deta Space (configs conflitantes)

## 6. PIPELINE DE PROSPECCAO (CORE DO PRODUTO)

### Fluxo Completo

1. EXTRACAO: extract.py -> Overpass API (OpenStreetMap)
   - Input: location, category, radius, limit
   - Output: JSON com leads (nome, telefone, endereco, tipo, coordenadas)
   - Funciona, testado com leads reais (Jacarepagua, Botafogo, Porto Seguro)

2. SINCRONIZACAO: pipeline.py post_to_dashboard() -> API /api/leads
   - Envia leads para o dashboard (JSON file ou Neon)
   - Depende de DASHBOARD_API_KEY configurada

3. GERACAO DE PROPOSTAS: run.py -> hermes CLI (Claude AI)
   - Q U E B R A D O - hermes CLI nao esta disponivel/instalado
   - Workaround: generate_proposals_direct.py (templates hardcoded)
   - Propostas sao textos de vendas para WhatsApp

4. OUTREACH: MANUAL
   - Pipeline gera propostas em .txt
   - Instrucao: "Copie os textos para WhatsApp" (pipeline.py linha 215)
   - SEM automacao de envio (Z-API, WPPConnect nao implementados)

5. FOLLOW-UP: MANUAL
   - Dashboard permite marcar status (contatado, negociacao, fechado)
   - Sem automacao de follow-up

### Onde o Fluxo Quebra

1. EXTRACAO -> SINCRONIZACAO: DASHBOARD_API_KEY precisa estar configurada
2. SINCRONIZACAO -> PROPOSTAS: hermes CLI nao existe -> fallback para template hardcoded
3. PROPOSTAS -> OUTREACH: 100% manual (copiar e colar no WhatsApp)
4. OUTREACH -> FOLLOW-UP: sem automacao

### Workflows n8n

NENHUM workflow versionado. Diretorio n8n-workflows/ contem apenas setup.py e py.typed.
